Figure.Collector
The Ultimate Digital Vault for Anime Assets
Figure.Collector — это не просто каталог, это высокотехнологичный реестр для энтузиастов и коллекционеров аниме-фигурок. Проект рожден из идеи превратить обычное хобби в структурированную базу данных с эстетикой киберпанка и строгим контролем активов.

🛠 Tech Stack Protocol
Система построена на надежных и современных узлах:

Frontend: React 18 с использованием Tailwind CSS для реализации кастомной темной темы и сложной сеточной разметки.

Backend / DB: Firebase (Firestore & Auth) — обеспечивает синхронизацию глобального каталога и безопасность личных данных в реальном времени.

Animations: Framer Motion для плавных переходов интерфейса и динамических состояний карточек.

Icons: Lucide React — минималистичный набор системных иконок.

External API: Интеграция с Jikan API (MyAnimeList) для автоматического подтягивания артов и данных персонажей.

📡 Core Modules
1. Personal Vault (My Shelf)
Индивидуальное пространство коллекционера.

Management: Полный CRUD (создание, чтение, обновление, удаление) ваших фигурок.

Smart Filters: Система «умной» калибровки списка по производителю, серии и ценовым категориям.

Status Tracking: Учет пола персонажа, состояния упаковки и рыночной стоимости.

2. Global Catalog (Community Hub)
Единая база данных всех пользователей системы.

Identity Sync: Каждая единица в каталоге имеет метку своего владельца.

Deep Search: Мгновенный поиск по всей глобальной базе активов.

Cross-Referencing: Возможность просматривать похожие фигурки из той же серии или от того же бренда.

3. Identity Card System
Каждая фигурка представлена в виде «Collector ID Card». Это уникальный интерфейс, вдохновленный пропусками в секретные лаборатории, где отображается полная техническая спецификация объекта.

🎨 Design Philosophy
Проект придерживается строгого визуального кода:

Base: #121212 (Deep Night)

Accent: Blue-500 (Electric Cyan)

Typography: Тяжелые начертания (Black), курсив (Italic) и увеличенный межбуквенный интервал (Tracking) для создания эффекта технической документации.
